# Code of Conduct

## Our commitment

e2hat is built on the belief that privacy is a right. This extends to how we treat each other. Everyone who participates in this project — whether by reporting issues, submitting code, or asking questions — deserves a respectful and welcoming environment.

## Expected behavior

- **Be respectful.** Disagree with ideas, not people.
- **Assume good intentions.** Most misunderstandings come from context, not malice.
- **Focus on the work.** Keep discussions relevant to the project.
- **Be constructive.** If you point out a problem, suggest a path forward when you can.
- **Be patient.** Not everyone has the same background or experience level. Take the time to explain.

## Unacceptable behavior

- Personal attacks, insults, or derogatory language
- Harassment of any kind, public or private
- Publishing others' private information without consent
- Sustained disruption of discussions
- Any conduct that would be inappropriate in a professional setting

## Scope

This code of conduct applies to all project spaces — issues, pull requests, discussions, and any other communication channel associated with e2hat.

## Enforcement

Instances of unacceptable behavior may be reported to the project maintainers. All reports will be reviewed and will result in a response appropriate to the circumstances. Maintainers are committed to maintaining confidentiality regarding the reporter.

Participants who violate this code of conduct may be temporarily or permanently excluded from the project.

## Attribution

This code of conduct is intentionally simple, like the project itself. It draws inspiration from the [Contributor Covenant](https://www.contributor-covenant.org/).
